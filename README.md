# BITEC BT06/07 2026 - JAVA-Unterlagen
Dieses Repository enthält sämtliche Unterlagen, die im Laufe des Kurses bereitgestellt werden. Die Materialien werden laufend aktualisiert und dienen als Ergänzung zu den Aufgaben in GitHub Classroom.

## 📁 Ordnerstruktur

- **code/**  
  Beispielprogramme, Erklär-Code und Demonstrationen aus dem Unterricht.

- **slides/**  
  Präsentationen und Folien als PDF.

- **extras/**  
  Zusätzliche Materialien wie Links, Cheatsheets, etc.

- **pages/**  
  Organisatorischer Ablauf des Unterrichts

## 🧭 Wie Sie dieses Repository verwenden

- Sie können jederzeit auf alle Materialien zugreifen.  
- Bei neuen Unterrichtseinheiten wird der passende Ordner aktualisiert.  
- Aufgaben liegen **nicht** hier, sondern werden über **GitHub Classroom** verteilt.  

## 💡 Hinweise

- Änderungen an den Materialien erscheinen sofort – Sie müssen nichts installieren.  
- Wenn Sie Dateiänderungen nicht sehen, aktualisieren Sie die Seite.
